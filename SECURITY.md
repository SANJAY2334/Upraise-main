# Security Policy

## Supported Versions

We actively support security updates for the following versions:

| Version  | Supported          |
| -------- | ------------------ |
| v1.0.0   | :white_check_mark: |
| < v1.0.0 | :x:                |

## Reporting a Vulnerability

We take the security of the UPRISE Platform seriously. If you find a security vulnerability, please do **NOT** open a public issue on GitHub. Instead, report it privately to our security team.

Please email security reports to **security@uprise.example**.

In your report, please include:

1. A descriptive title and description of the vulnerability.
2. Steps to reproduce or a Proof of Concept (PoC) script.
3. Potential impact or privilege escalation vectors.

We will acknowledge receipt of your vulnerability report within 48 hours and work with you to remediate it. We ask that you follow responsible disclosure principles and allow us up to 30 days to deploy patches before releasing any details publicly.
